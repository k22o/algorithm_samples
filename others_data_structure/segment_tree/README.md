# Data Strucuture

### 命名規則<br>
- RMQ(range minimum query)：最小値を求める
- RSQ(range sum query)：和を求める
- RUQ(range update query)：更新する
- RAQ(range add query)：追加する

- segment tree：1つずつの操作
- lazy segment tree：複数同時操作
- square division：平方分割
